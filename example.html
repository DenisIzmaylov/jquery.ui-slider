Little help
You can also specifiy custom options for your control:
  • vertical - orientation can be vertical or horizontal (by default)
  • smooth - to limit points only by specified values, by default is false
  • seekOnTrack - to allow seek by track line
  • seekOnOwner - to allow seek by whole component area
